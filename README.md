## Getting Started

This project require NodeJS v12

### Install
Clone this project

```bash
$cd  /sdcard
$cp -r saiki-bot  /$HOME
$cd
$cd saiki-bot
```

Install the dependencies:

```bash
$ pkg install nodejs
$ pkg install ffmpeg
$ pkg install imagemagick
$ bash install.sh
```

### Usage
Run the Whatsapp bot

```bash
$ node index.js
```
